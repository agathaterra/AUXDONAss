# 🏠 AUXDONAss - Auxílio de Controle Financeiro Doméstico

O **AUXDONAss** é uma aplicação desktop leve e intuitiva desenvolvida para auxiliar donas de casa e gestores domésticos no controle rigoroso de suas finanças. O projeto foca em transformar a complexidade de planilhas em uma interface visual simples, direta e funcional.

> **Status do Projeto:** 🚀 Versão Beta (Pronto para uso Desktop)

---

## 🌟 Por que o AUXDONAss?

A gestão do lar é uma das tarefas mais complexas que existem. O AUXDONAss foi criado para oferecer:
- **Clareza Imediata:** Saiba exatamente quanto entrou e quanto saiu no mês.
- **Saúde Financeira:** Visualize através de gráficos se você está economizando ou gastando além da conta.
- **Simplicidade:** Sem menus complicados. Tudo o que você precisa em uma única tela.

## ✨ Funcionalidades Principais

- **📊 Dashboard Dinâmico:** Gráfico de barras nativo que compara Entradas vs. Saídas em tempo real.
- **💰 Gestão de Fluxo:** Registro rápido de receitas e despesas com categorização simples.
- **💾 Persistência de Dados:** Salvamento automático em formato JSON (seus dados continuam lá ao fechar o programa).
- **🛠️ Controle Total:** Funções para excluir registros individuais ou resetar o histórico completo.
- **🎨 Interface Adaptiva:** Sistema de cores que alerta visualmente sobre o saldo (Verde para positivo, Vermelho para negativo).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído focando na **máxima performance e portabilidade**, utilizando apenas a biblioteca padrão do Python:

- **Python 3.10+**
- **Tkinter:** Para a interface gráfica (GUI).
- **JSON:** Para armazenamento local de dados.
- **Auto-py-to-exe:** Para geração do executável (.exe).
